À propos

Je suis en première année de BTS CIEL option A à Ozanam (Lille), en reconversion professionnelle. J'ai choisi cette formation parce que l'informatique et les réseaux m'attiraient vraiment — pas par défaut. Le programme couvre la cybersécurité, les réseaux, l'électronique et la programmation.

À l'école on utilise Debian, ce qui m'a donné envie d'aller plus loin sur Linux. J'utilise CachyOS + KDE comme environnement principal et je gère un homelab Docker — Ollama avec accélération GPU, Open WebUI, AdGuard Home, Tailscale. C'est là que j'apprends le plus concrètement.

Je réalise un stage de 6 semaines chez OVH Cloud (Roubaix, mi-mai à fin juin 2026) pour mettre en pratique ce que j'apprends, dans un environnement avec de vraies contraintes techniques.

Projets

01
Setup Linux — CachyOS + homelab Docker
Migration complète vers Linux comme OS principal. Configuration GPU NVIDIA RTX 3050 avec CUDA, gestion de services systemd, et montage d'un homelab Docker — Ollama, Open WebUI, AdGuard, Tailscale. Plusieurs itérations (EndeavourOS → CachyOS) qui m'ont appris comment Linux fonctionne vraiment.
Linux CachyOS Docker NVIDIA CUDA

02
Game Streaming — Sunshine / Moonlight
Juste pour voir si c'était faisable : streamer des jeux depuis mon PC Linux vers mon téléphone sur le réseau local. J'ai configuré Sunshine comme serveur et Moonlight comme client. Ca marche, et j'ai appris des choses sur les flux réseau au passage.
Sunshine Moonlight Réseau local Linux

03
Projets C++ — POO & Algorithmique
Exercices de programmation orientée objet réalisés en cours : gestion de mémoire dynamique, classes, constructeurs/destructeurs. Aussi quelques algos numériques comme l'estimation de π par Monte Carlo.
C++ POO Algorithmique
↗
04
IA locale — Ollama + Open WebUI + Docker
Stack d'IA entièrement en local : Ollama avec accélération GPU (RTX 3050, CUDA) pour faire tourner des LLMs, Open WebUI dans Docker comme interface, et Tailscale pour y accéder depuis n'importe où à distance.
Ollama Docker Tailscale Linux
↗
05
Infrastructure réseau — AdGuard + Vaultwarden + Monitoring
Stack de sécurité réseau maison déployée sur Docker : AdGuard Home comme DNS sinkhole (580k règles de filtrage, clients nommés par appareil), Vaultwarden (Bitwarden self-hosted) avec TLS mkcert, Uptime Kuma v2 pour le monitoring de disponibilité. Firewall iptables avec chaîne DOCKER-USER pour isoler les services.
AdGuard DNS Docker Sécurité iptables
↗
06
Backup automatisé — Kopia + systemd timers
Mise en place d'une stratégie de sauvegarde automatisée : Kopia (backup chiffré AES-256 + déduplication) déclenché chaque nuit par un timer systemd, rotation 14j/4sem/3mois. Backup Vaultwarden en parallèle via script Docker. Exploration de la gestion du cycle de vie des données.
Kopia systemd Bash Chiffrement Docker
↗
07
App révision BTS CIEL — Docker + Nginx + SQLite
Application web de révision auto-hébergée pour les examens BTS CIEL : fiches flash (51 cartes), exercices maths interactifs avec correction automatique, quiz Q&R (60 questions réseaux/C++/maths/culture), countdown dynamique. Stack : HTML/CSS/JS vanilla, Nginx, Docker Compose, JSON sync API.
JavaScript Docker Nginx HTML/CSS SQLite
